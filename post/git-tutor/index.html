Git Tutor: Interactive Git Tutorial

A standalone, offline-friendly tutorial website to help you practice Git basics with immediate feedback.

What you can do in Git Tutor:

  • Run Git commands in a terminal-like simulator.
  • Watch a live commit graph update as you create branches, commits, and merges.
  • Follow guided lessons and unlock small achievements.

Open Git Tutor

Tip: for best results when testing locally, run a simple server (e.g. python3 -m http.server 8000) and open /git_tutor/ from there.
