# Daily Issue (JST): 2026-05-14

## What's New Today

**AlphaGRPO, Pion, and Fast-Slow Learning: New Frontiers in LLM Post-Training and Memory**

Today's papers span AI/ML and materials physics, with a strong focus on advancing LLM post-training, memory, and reasoning. Key contributions include AlphaGRPO for self-reflective multimodal generation, Pion as a spectrum-preserving optimizer, and a fast-slow learning framework for continual adaptation. In materials science, new work on Coulombic contact interactions and automated phase identification from powder diffraction are highlighted.

**Highlights**
- AlphaGRPO applies GRPO to multimodal models for self-reflective generation and reasoning.
- Pion introduces a spectrum-preserving optimizer via orthogonal transformations for LLM training.
- Fast-slow learning framework combines in-context and in-weight adaptation for continual LLM learning.
- RADAR-PD automates multiphase identification in X-ray and neutron powder diffraction.
- Design rules for Coulombic contact interactions in polarizable particle systems are derived.
- KV-Fold enables training-free long-context inference via KV-cache recurrence.
- Attractor Models use implicit differentiation for stable, adaptive-depth recurrent language modeling.

### [AlphaGRPO: Unlocking Self-Reflective Multimodal Generation in UMMs via Decompositional Verifiable Reward](https://arxiv.org/abs/2605.12495v1)
**Authors:** Runhui Huang, Jie Wu, Rui Yang, Zhe Liu, Hengshuang Zhao
**Published:** 2026-05-12
**Summary:** AlphaGRPO applies GRPO to multimodal models for self-reflective generation without cold-start, using a decompositional verifiable reward.

**What's new**
- First framework to apply GRPO to AR-Diffusion UMMs without cold-start stage.
- Introduces Decompositional Verifiable Reward (DVReward) for stable, interpretable supervision.
- Enables reasoning text-to-image generation and self-reflective refinement in UMMs.
- Achieves robust gains on GenEval, TIIF-Bench, DPG-Bench, WISE, and editing tasks.

### [Pion: A Spectrum-Preserving Optimizer via Orthogonal Equivalence Transformation](https://arxiv.org/abs/2605.12492v1)
**Authors:** Kexuan Shi, Hanxuan Li, Zeju Qiu, Yandong Wen, Simon Buchholz, Weiyang Liu
**Published:** 2026-05-12
**Summary:** Pion is a spectrum-preserving optimizer for LLMs that updates weights via orthogonal transformations, keeping singular values fixed.

**What's new**
- Uses left and right orthogonal transformations to preserve singular values during training.
- Modulates weight matrix geometry while keeping spectral norm fixed.
- Provides a stable alternative to Adam and Muon for LLM pretraining and finetuning.

### [Learning, Fast and Slow: Towards LLMs That Adapt Continually](https://arxiv.org/abs/2605.12484v1)
**Authors:** Rishabh Tiwari, Kusha Sareen, Lakshya A Agrawal, Joseph E. Gonzalez, Matei Zaharia, Kurt Keutzer, Inderjit S Dhillon, Rishabh Agarwal, Devvrit Khatri
**Published:** 2026-05-12
**Summary:** Introduces Fast-Slow Training (FST) for LLMs, combining slow weight updates with fast in-context learning to improve sample efficiency and reduce forgetting.

**What's new**
- FST uses optimized context as fast weights and model parameters as slow weights.
- Up to 3x more sample-efficient than RL-only training on reasoning tasks.
- Reduces catastrophic forgetting by up to 70% less KL divergence from base model.
- Preserves plasticity, enabling better adaptation to new tasks in continual learning.

### [KV-Fold: One-Step KV-Cache Recurrence for Long-Context Inference](https://arxiv.org/abs/2605.12471v1)

**Published:** 2026-05-12

**What's new**

### [Beyond GRPO and On-Policy Distillation: An Empirical Sparse-to-Dense Reward Principle for Language-Model Post-Training](https://arxiv.org/abs/2605.12483v1)
**Authors:** Yuanda Xu, Hejian Sang, Zhengze Zhou, Ran He, Zhipeng Wang, Alborz Geramifard
**Published:** 2026-05-12
**Summary:** Scarce labeled data should train a strong teacher via sparse RL, then transfer to a student via dense supervision, not directly on the student.

**What's new**
- Proposes a sparse-to-dense reward principle: sparse RL for teacher exploration, dense supervision for student compression.
- Shows RL-improved teacher distilled via dense bridge outperforms direct GRPO on student at fixed size.
- Forward-KL warmup on teacher rollouts plus OPD on student rollouts gives strongest MATH results.
- Dense bridge makes later student-side sparse RL effective, lifting MATH from 75.4% to 78.5%.
- Avoid using scarce labeled data on the least prepared policy; allocate upstream for reward shaping.
