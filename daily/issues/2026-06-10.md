### [The Neutral Mask: How RLHF Provides Shallow Alignment while Leaving Partisan Structure Intact in a Large Language Model](https://arxiv.org/abs/2606.09735v1)
**Authors:** Wendy K. Tam
**Published:** 2026-06-08
**Summary:** RLHF provides shallow alignment by severing causal pathways from partisan structure to output, not removing the structure itself.

**What's new**
- RLHF compresses partisan signal variance for balanced output without erasing underlying geometry.
- Sparse autoencoder shows policy-encoding features inactive in Instruct model, active in base model.
- Feature-level steering confirms causal disconnect between partisan geometry and output generation.
- Neutrality is functional, not structural; partisan steering remains possible via user identity inference.
- RLHF's shallow alignment may make model behavior fragile across other value domains.
