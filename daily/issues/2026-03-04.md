# Daily Issue (JST): 2026-03-04

## What's New Today

**AI Reasoning Advances, Materials Discovery, and Efficient Systems Dominate Research**

Today's research spans efficient AI reasoning architectures, novel materials simulation methods, and robust systems for real-world deployment. Advances in symbolic data generation, conformal prediction, and test-time adaptation highlight progress in reliable and scalable AI. Meanwhile, high-accuracy frameworks for molecular crystals and electronic structure calculations push the boundaries of computational materials science.

**Highlights**
- Reasoning Core suite generates scalable, verifiable symbolic data for pre-training and post-training LLMs.
- Conformal policy control provides safety guarantees for agent exploration in high-stakes environments.
- Efficient first-principles framework achieves sub-chemical accuracy for complex molecular crystals.
- Tool-verification method (T^3RL) prevents incorrect mode collapse in test-time reinforcement learning.
- Symbol-equivariant recurrent reasoning models (SE-RRMs) solve puzzles like Sudoku with strong generalization.
- Multi-head low-rank attention (MLRA) reduces KV cache overhead for long-context inference.

## Featured Papers

### [Reasoning Core: A Scalable Procedural Data Generation Suite for Symbolic Pre-training and Post-Training](https://arxiv.org/abs/2603.02208v1)
**Authors:** Valentin Lacombe, Valentin Quesnel, Damien Sileo
**Published:** 2026-03-02
**Summary:** A scalable suite that procedurally generates verifiable symbolic reasoning data across formal domains like planning, logic, and grammar, improving model reasoning when mixed into pre-training.

**What's new**
- Generates verifiable symbolic data across five core formal domains: PDDL planning, first-order logic, grammar parsing, causal reasoning, and systems of equations
- Provides continuous difficulty control for curriculum design and pairs each task with an external solver for rigorous verification
- Includes optional solver-derived reasoning traces for supervised training from early pre-training stages and verifiable reward functions for RL
- Experiments show mixing its data into pre-training improves downstream reasoning while preserving language modeling quality

### [Partial Causal Structure Learning for Valid Selective Conformal Inference under Interventions](https://arxiv.org/abs/2603.02204v1)
**Authors:** Amir Asiaee, Kavey Aryan, James P. Long
**Published:** 2026-03-02
**Summary:** A method for selective conformal prediction under unknown interventions that learns partial causal structure to identify exchangeable calibration data, with a robustness guarantee for coverage.

**What's new**
- Provides a contamination-robust coverage theorem with an explicit bound g(δ,n) on coverage degradation from misclassified calibration examples.
- Introduces a partial causal learning task to estimate only binary descendant indicators needed for selective calibration, not the full graph.
- Proposes algorithms for descendant discovery via perturbation intersection patterns and approximate distance-to-intervention estimation.
- Validates the method on synthetic linear SEMs, maintaining ≥0.95 coverage under contamination, and on real CRISPRi genomic data.

### [Tool Verification for Test-Time Reinforcement Learning](https://arxiv.org/abs/2603.02203v1)

**Published:** 2026-03-02
**Summary:** T^3RL improves test-time reinforcement learning for large reasoning models by using external tool verification to filter out spurious consensus and produce more reliable training signals.

**What's new**
- Introduces tool verification to reward estimation, using evidence like code execution to upweight verified model rollouts.
- Replaces standard majority voting with verification-aware voting to produce more reliable pseudo-labels for online training.
- Demonstrates significant improvements over baseline TTRL across various math benchmarks, with larger gains on harder problems.
- Frames the approach as verified online data synthesis, positioning tool verification as a key stabilization mechanism for self-evolution.
