## Featured Papers

### [Goose: Anisotropic Speculation Trees for Training-Free Speculative Decoding](https://arxiv.org/abs/2604.02047v1)
**Authors:** Tao Jin, Phuong Minh Nguyen, Naoya Inoue
**Published:** 2026-04-02
**Summary:** GOOSE accelerates LLM inference by building anisotropic speculation trees that prioritize high-quality context-matched tokens in a deep chain, with wide branches for lower-quality alternatives.

**What's new**
- Introduces anisotropic trees that are asymmetric, using a deep chain for reliable tokens and wide branches for unreliable ones.
- Proves optimal tree structure when a large quality gap exists between token sources like n-gram matches and statistical predictions.
- Implements adaptive spine trees dynamically based on token acceptance rates, without requiring model training.
- Achieves 1.9-4.3x lossless speedup across five LLMs, outperforming balanced-tree baselines by 12-33%.

### [SKILL0: In-Context Agentic Reinforcement Learning for Skill Internalization](https://arxiv.org/abs/2604.02268v1)
**Authors:** Zhengxi Lu, Zhiyuan Yao, Jinyang Wu, Chengcheng Han, Qi Gu, Xunliang Cai, Weiming Lu, Jun Xiao, Yueting Zhuang, Yongliang Shen
**Published:** 2026-04-02
**Summary:** SKILL0 is an in-context reinforcement learning framework that internalizes agent skills into model parameters via a curriculum that progressively removes skill context, enabling zero-shot autonomous behavior.

**What's new**
- Proposes skill internalization into model parameters instead of inference-time retrieval to avoid noise and token overhead.
- Introduces a training curriculum that starts with full skill context and linearly withdraws it until zero-shot operation.
- Groups skills by category and renders them with interaction history into compact visual context for efficient learning.
- Uses a Dynamic Curriculum to evaluate and retain only skills from which the current policy still benefits.
- Achieves significant improvements over RL baselines while maintaining highly efficient context under 0.5k tokens per step.

### [AlloyVAE: A generative model for complex probabilistic field-to-field relationships in alloys](https://arxiv.org/abs/2604.02281v1)
**Authors:** Ningyu Yan, Zhuocheng Xie, Kai Guo, Yejun Gu, Huajian Gao, Yang Xiang
**Published:** 2026-04-02
**Summary:** AlloyVAE is a physics-informed generative model that learns probabilistic relationships between alloy microstructure and mechanical fields, enabling both forward prediction and inverse design.

**What's new**
- Learns full conditional distributions of mechanical fields from microstructural inputs, capturing one-to-many structure-property relationships.
- Incorporates learned smoothing operators and a self-consistency mechanism to ensure functional regularity and physical plausibility.
- Generates multiple physically consistent realizations of mechanical fields under identical input conditions.
- Supports inverse design by optimizing composition fields to achieve targeted mechanical responses.
- Extensible to coupled mappings involving eigenstrain, providing a scalable alternative to conventional simulations.

### [Theory of Lineshapes in Optical-Optical Double Resonance Spectroscopy](https://arxiv.org/abs/2604.02262v1)
**Authors:** Kevin K. Lehmann
**Published:** 2026-04-02
**Summary:** Analytical and numerical lineshapes for Optical-Optical Double Resonance spectroscopy reveal complex power broadening effects, including inhomogeneous broadening despite Lorentzian shapes and lower saturation power than expected.

**What's new**
- Derives lineshapes for arbitrary pump/probe strength using 3-level density matrix steady-state solutions.
- Shows probe spectrum is two Lorentzians with Autler-Townes splitting when Doppler broadening is negligible.

- Finds broadening differs for co- and counter-propagating fields and is largely inhomogeneous despite Lorentzian shape.
- Identifies saturation power ~4x lower than for bare probe transition, contradicting homogeneous width interpretation.
