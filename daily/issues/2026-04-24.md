### [AVISE: Framework for Evaluating the Security of AI Systems](https://arxiv.org/abs/2604.20833v1)
**Authors:** Mikko Lempinen, Joni Kemppainen, Niklas Raesalmi
**Published:** 2026-04-22
**Summary:** AVISE is a modular open-source framework for evaluating AI system security, demonstrated via automated jailbreak tests on language models.

**What's new**
- Introduces AVISE, a modular open-source framework for AI vulnerability identification and security evaluation.
- Extends the Red Queen attack with an Adversarial Language Model (ALM) for multi-turn jailbreak attempts.
- Develops a Security Evaluation Test (SET) with 25 cases and an Evaluation Language Model (ELM) achieving 92% accuracy.
- Evaluates nine recent language models, finding all vulnerable to the augmented Red Queen attack to varying degrees.

**Published:** 2026-04-22
**Summary:** Introduces gauge-equivariant graph neural networks for lattice gauge theories, embedding non-Abelian symmetry into message passing.

**What's new**
- Extends equivariant learning from global to local symmetries via matrix-valued, gauge-covariant features.
- Message passing implements gauge-covariant transport, enabling nonlocal correlations from local operations.
- Validated across pure gauge, gauge-matter, and dynamical regimes.

### [SWE-chat: Coding Agent Interactions From Real Users in the Wild](https://arxiv.org/abs/2604.20779v1)
**Authors:** Joachim Baumann, Vishakh Padmakumar, Xiang Li, John Yang, Diyi Yang, Sanmi Koyejo
**Published:** 2026-04-22
**Summary:** First large-scale dataset of real coding agent sessions from open-source developers, revealing usage patterns and inefficiencies.

**What's new**
- Dataset contains 6,000 sessions with 63,000+ prompts and 355,000 tool calls from real users.
- Only 44% of agent-written code survives into user commits; agent code introduces more security vulnerabilities.
- Users push back against agent outputs in 44% of all turns via corrections, failures, or interruptions.
- Coding patterns are bimodal: 41% 'vibe coding' (agent writes all code) vs. 23% human-only writing.

### [ParetoSlider: Diffusion Models Post-Training for Continuous Reward Control](https://arxiv.org/abs/2604.20816v1)

**Published:** 2026-04-22
**Summary:** ParetoSlider trains a single diffusion model to approximate the entire Pareto front, enabling continuous control over conflicting rewards at inference time.

**What's new**
- Multi-objective RL post-training for diffusion models with continuous preference conditioning.
- Single model approximates entire Pareto front, eliminating need for multiple checkpoints.
- Enables fine-grained inference-time trade-off control between competing goals.
- Matches or exceeds baselines trained separately for fixed reward trade-offs.

### [Stream-CQSA: Avoiding Out-of-Memory in Attention Computation via Flexible Workload Scheduling](https://arxiv.org/abs/2604.20819v1)
**Authors:** Yiming Bian, Joshua M. Akey
**Published:** 2026-04-22
**Summary:** Stream-CQSA enables exact attention on billion-token sequences within arbitrary memory budgets via flexible workload scheduling.

**What's new**
- Introduces CQS Divide, a decomposition of attention into independent subsequence computations.
- Proposes Stream-CQSA, a memory-adaptive scheduling framework for attention.
- Eliminates the assumption that full Q, K, V tensors fit in device memory.
- Enables exact attention on a single GPU via streaming without approximation error.
- Demonstrates predictable memory scaling for billion-token sequences.
