# Daily Issue (JST): 2026-07-02

## What's New Today

**AI Agents Learn to Introspect, Adapt, and Compose Skills Across Domains**

Today's papers reveal a maturing AI landscape where models are learning to introspect on their own reasoning, adapt to distribution shifts at test time, and compose skills for complex tasks. Key advances include self-explanation training that yields faithful introspection, adaptive world models for robotic control, and frameworks for composing modular skills in LLM agents. The set also features significant work in materials physics, including new methods for simulating relativistic hydrodynamics and understanding molecular spin control.

**Highlights**
- Introspective Coupling: LMs trained on fixed counterfactual explanations often produce explanations more faithful to their own behavior than to training targets.
- AdaJEPA: An adaptive latent world model that performs test-time adaptation within model predictive control, improving planning under distribution shift.
- Generative Skill Composition: A framework for LLM agents that selects and composes modular skills, addressing a key bottleneck in complex task execution.
- TRIAGE: Role-typed credit assignment for agentic RL, using a structured judge to classify action segments and improve credit assignment.
- FLORA: A deep learning framework predicting forest attributes from heterogeneous LiDAR data, enabling national-scale resource monitoring.
- GQL-Based PCP Schemes: A robust framework for special relativistic hydrodynamics that preserves physical constraints using geometric quasilinearization.
- Conical Intersections Enable Ultrafast Molecular Spin Control: Direct observation of spin-state dynamics in a chromium complex, with implications for molecular spintronics.

## Featured Papers

### [Introspective Coupling: Self-Explanation Training Tracks Behavioral Change Despite Fixed Supervision](https://arxiv.org/abs/2606.32038v1)
**Authors:** Zifan Carl Guo, Laura Ruis, Jacob Andreas, Belinda Z. Li
**Published:** 2026-06-30
**Summary:** Training LMs on fixed counterfactual explanations can yield faithful introspection that tracks behavioral changes.

**What's new**
- LMs trained on fixed explanations from earlier checkpoints produce explanations more faithful to their own current behaviors.
- Introspective coupling occurs when training explanations remain correlated with current behaviors as behaviors shift.
- Explanation training tracks behavior shifts from other post-training objectives without updated supervision.
- Phenomenon is robust to label noise and appears in tasks like sycophancy and refusal.

### [AdaJEPA: An Adaptive Latent World Model](https://arxiv.org/abs/2606.32026v1)
**Authors:** Ying Wang, Oumayma Bounou, Yann LeCun, Mengye Ren
**Published:** 2026-06-30
**Summary:** AdaJEPA adapts a latent world model at test time via self-supervised closed-loop updates to improve planning under distribution shift.

**What's new**
- Proposes test-time adaptation of latent world models within MPC closed loop.
- Uses observed next-state transitions as self-supervised adaptation signals.
- Requires no expert demonstrations for recalibration.
- Achieves substantial planning improvement with as few as one gradient step per replan.

### [TRIAGE: Role-Typed Credit Assignment for Agentic Reinforcement Learning](https://arxiv.org/abs/2606.32017v1)

**Published:** 2026-06-30

**What's new**

### [GQL-Based Physical-Constraint-Preserving High-Order Finite Difference Schemes for Special Relativistic Hydrodynamics in Arbitrary Dimensions](https://arxiv.org/abs/2606.31992v1)
**Authors:** Linfeng Xu, Shengrong Ding, Kailiang Wu
**Published:** 2026-06-30
**Summary:** A robust PCP flux-limiting framework for high-order RHD schemes using GQL and FCT, with explicit non-iterative parameter determination via eigenvalue problems.

**What's new**
- Integrates Zalesak-type FCT into a scalar-style limiter acting on conservative variables.

- Introduces relaxed variants to reduce multidimensional costs while preserving PCP guarantee.
- Demonstrates robust enforcement of physical admissibility and design-order accuracy in 1D-3D benchmarks.

### [SemRF: A Semantic Reference Frame for Residual-Stream Dynamics in Language Models](https://arxiv.org/abs/2606.32022v1)
**Authors:** Jian Gu, Aldeida Aleti, Chunyang Chen, Hongyu Zhang
**Published:** 2026-06-30
**Summary:** Introduces Semantic Reference Frames (SemRF) to separate semantic measurement from residual dynamics in language models, enabling stable analysis of computation across layers.

**What's new**
- SemRF uses anchor-based formalism to fix coordinates, eliminating measurement drift from embedding-unembedding misalignment.
- Pseudo-inverse tying ensures exact synchronization; under bi-invertibility, yields stable semantic-basis coordinates with distortion bounds.
- Anchors induce a semantic Voronoi diagram, assigning layers to coarse cells while retaining within-cell motion and margins.
- Defines a canonical trace as the minimum-action path inside a margin-relaxed tube, unique under positive quadratic weight and obeying a discrete spline equation.
- Links trace complexity to parameter efficiency: lower-action, lower-complexity traces use fewer semantic degrees of freedom.
