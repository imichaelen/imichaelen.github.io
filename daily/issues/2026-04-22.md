## Featured Papers

### [MathNet: a Global Multimodal Benchmark for Mathematical Reasoning and Retrieval](https://arxiv.org/abs/2604.18584v1)
**Authors:** Shaden Alshammari, Kevin Wen, Abrar Zainal, Mark Hamilton, Navid Safaei, Sultan Albarakati, William T. Freeman, Antonio Torralba
**Published:** 2026-04-20
**Summary:** MathNet is a large-scale, multilingual, multimodal benchmark of Olympiad-level math problems for evaluating mathematical reasoning in generative models and retrieval in embedding systems.

**What's new**
- Introduces the largest high-quality Olympiad dataset with 30,676 expert problems across 47 countries and 17 languages
- Provides the first benchmark for evaluating mathematical problem retrieval, including equivalent and similar problem pairs
- Supports three tasks: Problem Solving, Math-Aware Retrieval, and Retrieval-Augmented Problem Solving
- Shows state-of-the-art models remain challenged, with GPT-5 at 69.3% and embedding models struggling with retrieval
- Demonstrates retrieval-augmented generation gains up to 12% for models like DeepSeek-V3.2-Speciale

### [Sessa: Selective State Space Attention](https://arxiv.org/abs/2604.18580v1)
**Authors:** Liubomyr Horbatko
**Published:** 2026-04-20
**Summary:** Sessa is a decoder architecture that combines attention with a feedback path, enabling recurrent multi-path aggregation for improved long-range power-law memory.

**What's new**
- Places attention inside a feedback path for recurrent many-path aggregation within a single layer
- Achieves power-law memory tail O(ℓ^{-β}) for 0<β<1, slower decay than 1/ℓ
- Enables flexible selective retrieval including non-decaying memory profiles
- Outperforms Transformer and Mamba baselines on long-context benchmarks while remaining competitive on short-context tasks

### [Bounded Ratio Reinforcement Learning](https://arxiv.org/abs/2604.18578v1)
**Authors:** Yunke Ao, Le Chen, Bruce D. Lee, Assefa S. Wahd, Aline Czarnobai, Philipp Fürnstahl, Bernhard Schölkopf, Andreas Krause
**Published:** 2026-04-20
**Summary:** BRRL is a new reinforcement learning framework with a theoretically grounded policy optimization algorithm (BPO) that ensures monotonic improvement, outperforming PPO in stability and performance across diverse tasks.

**What's new**
- Introduces Bounded Ratio RL (BRRL), a framework bridging trust region theory and practical algorithms like PPO.
- Derives an analytic optimal policy solution for a regularized/constrained problem, guaranteeing monotonic improvement.
- Proposes Bounded Policy Optimization (BPO), minimizing divergence from the analytic solution with performance lower bounds.
- Extends to Group-relative BPO (GBPO) for LLM fine-tuning, matching or outperforming GRPO and PPO in experiments.
- Provides new theoretical interpretation for PPO's success and connects trust region methods to Cross-Entropy Method.

### [Agentic Forecasting using Sequential Bayesian Updating of Linguistic Beliefs](https://arxiv.org/abs/2604.18576v1)
**Authors:** Kevin Murphy
**Published:** 2026-04-20
**Summary:** BLF is an agentic forecasting system that uses Bayesian linguistic belief states, multi-trial aggregation, and hierarchical calibration to achieve state-of-the-art performance on binary forecasting benchmarks.

**What's new**
- Introduces a Bayesian linguistic belief state, a semi-structured representation combining probabilities and evidence summaries updated iteratively.
- Employs hierarchical multi-trial aggregation with logit-space shrinkage using a data-dependent prior to combine independent trials.
- Uses hierarchical calibration with Platt scaling to avoid over-shrinking extreme predictions for sources with skewed base rates.
- Outperforms top public methods like Cassi and GPT-5 on 400 ForecastBench questions, with ablation showing key component contributions.
- Develops a robust back-testing framework with low leakage and rigorous statistical methodology for method comparison.

### [Latent Phase-Shift Rollback: Inference-Time Error Correction via Residual Stream Monitoring and KV-Cache Steering](https://arxiv.org/abs/2604.18567v1)
**Authors:** Manan Gupta, Dhruv Kumar
**Published:** 2026-04-20
**Summary:** LPSR corrects LLM reasoning errors at inference time by detecting abrupt changes in the residual stream and steering generation via KV-cache rollback, without fine-tuning or extra forward passes.

**What's new**
- Detects reasoning errors via a dual gate monitoring cosine-similarity and entropy in a critical layer's residual stream.
- Corrects errors by rolling back the KV-cache and injecting a pre-computed steering vector, requiring no fine-tuning.
- Boosts MATH-500 accuracy by +15.2 percentage points over standard autoregressive generation for an 8B model.
- Outperforms prompted self-correction by +24.2 pp and Best-of-16 sampling with 5.4x lower token cost.
- Reveals a detection-correction dissociation: optimal layers for error detection and task correction differ.
