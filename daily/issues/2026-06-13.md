## What's New Today

**AI Agents Evolve: From Dynamic Memory to Scientific Discovery**

Today's research spans agentic AI, materials physics, and foundational ML theory. Key advances include frameworks for robust LLM agents in dynamic environments, a new paradigm for autonomous scientific discovery via environment engineering, and novel insights into the geometry of phase transitions in generative models. Materials science contributions feature breakthroughs in chiral perovskite electronics and spin-polarized photoelectrochemistry.

**Highlights**
- EvoArena benchmarks LLM agents in dynamic environments; EvoMem patch-based memory improves adaptation.

- First robust lateral device integration in chiral 2D perovskite R-(MBA)2PbI4 enables Hall measurements.

- Operadic consistency (OC) provides a label-free signal for detecting compositional reasoning failures in LLMs.
- MaxProof achieves human gold-medal-level performance on IMO 2025 and USAMO 2026 via population-level test-time scaling.

### [Operadic consistency: a label-free signal for compositional reasoning failures in LLMs](https://arxiv.org/abs/2606.13649v1)
**Authors:** Nathaniel Bottman, Yinhong Liu, Kyle Richardson
**Published:** 2026-06-11
**Summary:** Operadic consistency (OC) detects LLM compositional reasoning failures by checking if direct answers match answers from stated decompositions, outperforming baselines.

**What's new**
- OC correlates strongly with accuracy across 12 LLMs on 4 datasets (Pearson r ≥ 0.86).
- OC uniquely achieves r ≥ 0.85 uniformly on all datasets, unlike chain-of-thought self-consistency.
- OC provides complementary information beyond CoT-SC and semantic entropy (p ≤ 10⁻¹⁶).
- OC improves selective prediction over tuned CoT-SC at equal cost (AUARC lifts +0.086 to +0.096).
- On frontier models, OC yields positive selective-prediction lift in all 16 tested cells.
