# Daily Issue (JST): 2026-06-12

## What's New Today

**Reroute, Don't Remove: Recoverable Visual Token Routing for VLMs**

Vision-language models (VLMs) project images into hundreds to thousands of visual tokens, making decoder inference expensive. Existing visual-token reduction methods follow a rank-and-remove paradigm, but this is fragile because token importance changes across decoder depth. Reroute, a training-free plug-in, replaces removal with recoverable routing, allowing deferred tokens to re-enter later.

**Highlights**
- Reroute is a training-free plug-in that replaces irreversible token removal with recoverable routing.
- Visual-token importance changes across decoder depth; low-ranked tokens may become relevant later.
- Deferred tokens bypass a stage and re-enter the candidate pool, improving grounding-sensitive queries.
- The method is designed as a drop-in replacement for existing rank-and-remove approaches.
- Reroute addresses a key fragility in current VLM token reduction strategies.
