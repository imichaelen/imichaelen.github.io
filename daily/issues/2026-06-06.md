# Daily Issue (JST): 2026-06-06

## What's New Today

**Daily Research Digest: LLM Agents, Physics, and Efficient ML**

Today's papers span LLM agent memory and safety, physics-informed ML, and efficient model architectures. Highlights include a new framework for agent memory characterization, a superconducting triode effect, and a method for pretraining RNNs without recurrence. The set also features work on continual learning, code generation, and autonomous driving safety.

**Highlights**
- Agent Memory: First systems characterization of stateful long-horizon LLM agent workloads, revealing cost attribution across construction, retrieval, and generation.
- Superconducting Triode: A quantum-dot Josephson junction with a biased top gate enables a continuously tunable unidirectional supercurrent, preserving time-reversal symmetry.
- RNN Pretraining without Recurrence: Supervised Memory Training (SMT) reduces RNN training to one-step supervised learning, sidestepping BPTT's parallelism and gradient issues.
- TailLoR for Continual Learning: A spectral penalty discourages updates aligned with dominant singular directions, reducing interference in parameter-efficient fine-tuning.
- Code2LoRA: Hypernetwork-generated LoRA adapters inject repository-level knowledge into code LMs with zero inference-time token overhead.
- RiskFlow for Autonomous Driving: A diffusion-based framework generates safety-critical traffic scenarios with reduced jitter and off-road artifacts via action-space transport.
- PC Layer for LLM Pre-training: Polynomial weight preconditioning stabilizes singular-value spectra, improving Llama-1B training with AdamW and Muon optimizers.

## Featured Papers

### [TailLoR: Protecting Principal Components in Parameter-Efficient Continual Learning](https://arxiv.org/abs/2606.06494v1)
**Authors:** Marius Dragoi, Ioana Pintilie, Alexandra Dragomir, Antonio Barbalau, Florin Brad
**Published:** 2026-06-04
**Summary:** TailLoR protects principal components in continual learning by routing updates to long-tail spectral coordinates.

**What's new**
- Uses fixed singular bases U and V from pre-trained weights as a reference frame.
- Learns low-rank updates applied to the singular value matrix.
- Soft spectral penalty discourages updates aligned with dominant singular directions.
- Reduces interference and routes adaptation to flexible long-tail spectral coordinates.

### [Code2LoRA: Hypernetwork-Generated Adapters for Code Language Models under Software Evolution](https://arxiv.org/abs/2606.06492v1)
**Authors:** Liliana Hotsko, Yinxi Li, Yuntian Deng, Pengyu Nie
**Published:** 2026-06-04
**Summary:** Code2LoRA uses a hypernetwork to generate repository-specific LoRA adapters for code LMs, eliminating inference token overhead and adapting to software evolution.

**What's new**
- Introduces Code2LoRA-Static for stable codebases and Code2LoRA-Evo for evolving codebases with GRU-based hidden state updates.
- Builds RepoPeftBench, a benchmark of 604 Python repositories with static and evolution tracks for evaluating parameter-efficient fine-tuning.
- Achieves 63.8% cross-repo exact match on static tasks, matching per-repository LoRA, and 60.3% on evolution tasks (+5.2 pp over shared LoRA).

### [Pretraining Recurrent Networks without Recurrence](https://arxiv.org/abs/2606.06479v1)
**Authors:** Akarsh Kumar, Phillip Isola
**Published:** 2026-06-04
**Summary:** SMT trains RNNs via supervised learning on memory transitions, avoiding BPTT's sequential and gradient issues.

**What's new**
- Reduces RNN training to supervised learning on one-step memory labels.
- Uses a Transformer encoder to generate memory labels via predictive state objective.
- Enables time-parallel training with O(1) gradient path between tokens.
- Outperforms BPTT on language and pixel sequence modeling tasks.
- Unlocks scaling of RNNs with better long-range dependency capture.

### [Superconducting triode effect in a quantum-dot Josephson junction with a biased top gate](https://arxiv.org/abs/2606.06465v1)
**Authors:** Yu-Hang Li, Xiaan Du, Hua Jiang, X. C. Xie
**Published:** 2026-06-04
**Summary:** A superconducting triode effect is proposed in a quantum-dot Josephson junction with a biased top gate, enabling tunable non-reciprocal supercurrents.

**What's new**
- Proposes a superconducting triode effect that breaks parity but preserves time-reversal symmetry.
- Demonstrates continuous gate control of non-reciprocal supercurrent, achieving ideal unidirectional flow.
- Shows highly asymmetric Shapiro steps under RF radiation, enabling quantized supercurrent rectification.
- Offers a new mechanism for the superconducting diode effect in time-reversal-symmetric junctions.

### [Nonreversible Gauge Fields in Fokker--Planck Dynamics: Supersymmetric Hamiltonians and Learned Finite Forces](https://arxiv.org/abs/2606.06412v1)
**Authors:** Masayuki Ohzeki
**Published:** 2026-06-04
**Summary:** Nonreversible gauge fields in Fokker-Planck dynamics preserve the stationary state while modifying relaxation spectra, with applications to learning finite forces.

**What's new**
- Formulates nonreversible perturbations as gauge fields that deform relaxation spectra while fixing the invariant state.
- Maps reversible Fokker-Planck operators to supersymmetric Hamiltonians; nonreversible gauges appear as non-Hermitian perturbations.

- Introduces a finite-time regularized objective and actor-critic procedure to learn the optimal gauge strength.
- Shows stochastic gradient methods (e.g., Adam) correspond to Fokker-Planck systems with nonequilibrium currents.
