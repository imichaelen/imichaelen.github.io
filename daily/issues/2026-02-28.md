# Daily Issue (JST): 2026-02-28

## What's New Today

**AI Agents Advance in Reasoning and Efficiency, Materials Science Explores Novel Synthesis and Properties**

Today's research highlights significant progress in AI agent frameworks, with new methods for improving reasoning diversity, memory efficiency, and training stability. In parallel, materials science advances include novel synthesis techniques for 2D materials and insights into thermal and magnetic properties. Cross-domain work also emerges, applying AI to discover physical laws and optimize materials design.

**Highlights**
- ParamMem introduces parametric memory to enhance reflection diversity in language agents, improving task success.
- FlashOptim reduces per-parameter memory by over 50% for efficient large-model training.
- MBE growth achieves wafer-scale, high-quality ultrathin SnSe van der Waals layers.
- A semi-supervised training-by-proxy method slashes data needs for generative models in magnetic material design.
- SOTAlign enables vision-language model alignment with limited paired data using optimal transport.
- Residual Koopman Spectral Profiling predicts transformer training instability from a single forward pass.
- A decision-theoretic framework formalizes steganography detection in LLMs without a reference distribution.

### [Model Agreement via Anchoring](https://arxiv.org/abs/2602.23360v1)
**Authors:** Eric Eaton, Surbhi Goel, Marcel Hussing, Michael Kearns, Aaron Roth, Sikata Bela Sengupta, Jessica Sorrell
**Published:** 2026-02-26
**Summary:** A general anchoring technique proves that independent training of common ML algorithms can drive model disagreement to zero as key parameters increase.

**What's new**
- Introduces anchoring technique to bound disagreement between independently trained models via their average.
- Proves disagreement vanishes for stacked aggregation as number of stacked models increases.
- Shows gradient boosting disagreement goes to zero with more iterations.
