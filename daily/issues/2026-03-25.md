### [WorldCache: Content-Aware Caching for Accelerated Video World Models](https://arxiv.org/abs/2603.22286v1)
**Authors:** Umair Nawaz, Ahmed Heakl, Ufaq Khan, Abdelrahman Shaker, Salman Khan, Fahad Shahbaz Khan
**Published:** 2026-03-23
**Summary:** WorldCache accelerates video diffusion model inference via a perception-constrained dynamical caching framework that adaptively reuses features to reduce computation while maintaining quality.

**What's new**
- Introduces motion-adaptive thresholds and saliency-weighted drift estimation to decide when to reuse cached features.
- Proposes optimal feature approximation via blending and warping to improve how features are reused, reducing artifacts.
- Implements phase-aware threshold scheduling across diffusion steps for adaptive caching behavior.
- Achieves 2.3x inference speedup on a 2.5B parameter model while preserving 99.4% of baseline output quality.

### [Scaling DoRA: High-Rank Adaptation via Factored Norms and Fused Kernels](https://arxiv.org/abs/2603.22276v1)
**Authors:** Alexandra Zelenin, Alexandra Zhuravlyova
**Published:** 2026-03-23
**Summary:** A memory-efficient, high-performance implementation of DoRA (Weight-Decomposed Low-Rank Adaptation) using factored norm computations and fused GPU kernels.

**What's new**
- Factored norm computation eliminates the need to materialize the dense BA product, reducing transient memory from ~512MB to O(d_out r + r^2).
- Fused Triton kernels combine the four-kernel DoRA composition into a single pass, cutting memory traffic by ~4x and improving numerical stability.
- Achieves 1.5-2.7x speedup over Hugging Face PEFT's DoRA implementation for inference and gradient computation across various GPUs.
- Reduces peak VRAM usage by up to 7 GB while maintaining high numerical fidelity (cosine similarity >0.9999) and matching training curves.
