Today's papers span AI/ML advances and materials physics, with highlights including the first cross-architecture distillation framework for diffusion LLMs, a new altermagnetic transport fingerprint, and an uncertainty-aware safety filter for RL. ### [Turning the TIDE: Cross-Architecture Distillation for Diffusion Large Language Models](https://arxiv.org/abs/2604.26951v1)
**Authors:** Gongbo Zhang, Wen Wang, Ye Tian, Li Yuan
**Published:** 2026-04-29
**Summary:** TIDE enables cross-architecture distillation for diffusion LLMs, boosting a 0.6B student with 8B/16B teachers.

**What's new**
- First framework for cross-architecture dLLM distillation with different architectures and tokenizers.
- TIDAL modulates distillation strength based on training progress and diffusion timestep.
- CompDemo enriches teacher context via complementary mask splitting for better heavy-masking predictions.
- Reverse CALM inverts chunk-level likelihood matching for bounded gradients and noise filtering.
- Outperforms baseline by 1.53 points on 8 benchmarks; HumanEval rises from 32.3 to 48.78.

### [A self-evolving agent for explainable diagnosis of DFT-experiment band-gap mismatch](https://arxiv.org/abs/2604.26703v1)

**Published:** 2026-04-29
**Summary:** XDFT is a self-evolving agent that automatically diagnoses DFT-experiment band-gap mismatches with 78% accuracy.

**What's new**
- Closed-loop agent iteratively tests hypotheses from a curated catalogue using first-principles calculations.
- Achieves 78% resolution on 124-material benchmark, far outperforming random (19%) and static LLM (20%) baselines.
